Traducteur Français-Alsacien

    VOYELLES

  • a ou aa => a
  • à ou àà => o ouvert
  • ä ou ää => ai
  • é ou éé => é
  • i => ie
  • ì => entre 'é' et 'i'
  • o ou oo => o fermé
  • ö ou öö => eu
  • ù => entre 'ou' et 'on'
  • ü o üü => u

    • DIPHTONGUES

  • ai => aïe
  • au => aou
  • äu => oeil
  • ei => aïl
  • ej => éyi
  • ue ou üe => euh
  • ch => ch, mais 'r' dur après a, à, o, ù

    • CONSONNES

  • h => expiré
  • j => y
  • qu => k
  • s => ss
  • sch => sch (schéma)
  • sp => ch'p
  • st => ch't
  • v => f
  • w => v
  • z => ts
